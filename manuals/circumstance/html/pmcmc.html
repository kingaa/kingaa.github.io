pmcmc {circumstance}R Documentation

Particle Markov chain Monte Carlo in parallel

Description

Runs multiple instances of pmcmc using foreach.

Usage

## S4 method for signature 'ANY,data.frame'
pmcmc(data, starts, ...)

## S4 method for signature 'ANY,missing'
pmcmc(data, starts, ...)

## S4 method for signature 'pompList,missing'
pmcmc(data, starts, ...)

## S4 method for signature 'pfilterList,missing'
pmcmc(data, starts, ...)

## S4 method for signature 'pmcmcList,missing'
pmcmc(data, starts, ...)

Arguments

data

passed to pomp::pmcmc

starts

data frame containing parameters at which to begin iterated filtering

...

all additional arguments are passed to pomp::pmcmc

See Also

pomp::pmcmc.
