Diagramming internals

Description

Facilities to produce diagrammatic representations of genealogy process states.

Usage

Arguments

object

list; pocket structure

m

width of plotting window, in nodes. By default, the nodes will be adjusted in width to fit the window.

n

length of longest genealogy

vp

viewport to draw plot in

...

arguments to be passed to textGrob.

digits

non-negative integer; number of decimal digits to print in the node time

x

An R object.

recording

A logical value indicating whether a grob is being added to the display list or redrawn from the display list.

Details

Code for the resizing text adapted from a blog post by Mark Heckmann (https://ryouready.wordpress.com/2012/08/01/creating-a-text-grob-that-automatically-adjusts-to-viewport-size/).
