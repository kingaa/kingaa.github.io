parse_newick {phylopomp}R Documentation

parse a Newick string

Description

Parses a Newick description and returns a binary version of the genealogy.

Usage

parse_newick(x, t0 = 0, tf = NA)

Arguments

x

character; the Newick description. See Details for specifics.

t0

numeric; the root time.

tf

numeric; the current or final time.

Value

An object of class “gpgen”.
