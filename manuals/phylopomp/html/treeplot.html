treeplot {phylopomp}R Documentation

Fancy tree plotter

Description

Plots a genealogical tree.

Usage

treeplot(
  tree,
  time = NULL,
  illus = NULL,
  root_time = 0,
  ladderize = TRUE,
  points = FALSE,
  diagram = FALSE
)

Arguments

tree

character; tree representation in Newick format.

time

numeric; times of the genealogies.

illus

character; genealogy process diagram information.

root_time

numeric; time of the root.

ladderize

Ladderize?

points

Show nodes and tips?

diagram

Show a diagram?

Value

A printable ggtree object.
