Comparison of particle-filter likelihood with exact formula for the linear birth-death-sampling process

A.A. King

The following codes simulate a realization of the genealogy process induced by a linear birth-death-sampling process with birth rate \(\lambda\), death rate \(\mu\) and sampling rate \(\psi\).

runLBDP(lambda=1.5,mu=0.8,psi=1,time=5,n0=5) -> x
x |> lineages() -> lbdp_dat

The following designs a slice through parameter space in the \(\lambda\) direction. In addition, three different levels of computational effort are applied (as measured by the number of particles, Np).

expand_grid(
  rep=1:4,
  lambda=seq(0.7,2.5,by=0.1),
  mu=0.8,
  psi=1,
  time=5,
  n0=5,
  Np=c(1000,10000,100000)
) -> params

Next, we perform a sequential Monte Carlo (particle filter) computation at each point in the above design.

library(doParallel)
library(doRNG)
registerDoParallel()
registerDoRNG(728604304)
foreach (
  p=iter(params,"row"),
  .inorder=FALSE,
  .packages=c("magrittr","phylopomp","pomp"),
  .combine=bind_rows
) %dopar% {
  p %$% {
    lbdp_dat |>
      lbdp_exact(lambda=lambda,mu=mu,psi=psi,n0=n0)
  } -> ll1
  p %$% {
    lbdp_dat |>
      lbdp_pomp(lambda=lambda,mu=mu,psi=psi,n0=n0) |>
      pfilter(Np=Np) |>
      logLik()
  } -> ll2
  bind_cols(p,exact=ll1,pf=ll2)
}-> params
