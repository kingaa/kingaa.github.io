pred.var {pomp}R Documentation

Prediction variance

Description

The variance of the prediction distribution

Usage

## S4 method for signature 'pfilterd_pomp'
pred.var(object, vars, ...)

Arguments

object

result of a filtering computation

vars

optional character; names of variables

...

ignored

Details

The prediction distribution is that of

Xk | Y1=y1*,…,Y(k-1)=y(k-1)*,

where Xk, Yk are the latent state and observable processes, respectively, and yk* is the data, at time tk.

The prediction variance is therefore the variance of this distribution

Var[Xk | Y1=y1*,…,Y(k-1)=y(k-1)*].
