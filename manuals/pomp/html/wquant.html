wquant {pomp}R Documentation

Weighted quantile function

Description

Computes weighted quantiles.

Usage

wquant(
  x,
  weights,
  probs = c(`0%` = 0, `25%` = 0.25, `50%` = 0.5, `75%` = 0.75, `100%` = 1)
)

Arguments

x

numeric; a vector of data.

weights

numeric; vector of weights.

probs

numeric; desired quantiles.

Details

wquant computes a quantile of type 7 according to the typology of quantile.

Value

wquant returns a vector containing the quantiles. If probs has names, these are inherited.

Author(s)

Aaron A. King

Examples

x <- c(1,1,1,2,2,3,3,3,3,4,5,5,6,6,6)
quantile(x)
wquant(c(1,2,3,4,5,6),weights=c(3,2,4,1,2,3))
